# Joe Gutierrez Personal Website

Static personal portfolio website built for GitHub Pages.

## How to upload to GitHub

1. Create a GitHub repository named `YOUR-USERNAME.github.io`.
2. Upload everything inside this folder, not the ZIP itself.
3. Make sure `index.html`, `styles.css`, and the `assets` folder are at the top/root level.
4. Go to **Settings > Pages**.
5. Under **Branch**, choose `main` and `/root`, then click **Save**.
6. Your site will publish at `https://YOUR-USERNAME.github.io`.

## Files

- `index.html` — website content
- `styles.css` — styling
- `assets/joe-gutierrez-portrait.png` — hero photo
- `assets/Joe-Gutierrez-Resume.pdf` — downloadable resume
